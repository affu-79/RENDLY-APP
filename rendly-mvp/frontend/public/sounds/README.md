# Call ringtones

Add loopable ringtone audio files here for incoming calls:

- **ringtone.mp3** – played for incoming video calls
- **ringtone-audio.mp3** – played for incoming audio calls

Use short, royalty-free loopable clips (e.g. 2–4 seconds). If files are missing, the app will still work; ringtone playback will fail silently.
